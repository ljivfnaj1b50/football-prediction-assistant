# 鲸喜足球模型 V2

本仓库已从上一版手动录入结构，调整为 V2 自动数据分析框架。

## 当前状态

已更新：

- `index.html`：新版仪表盘页面
- `styles.css`：新版深色专业界面
- `model.js`：赛事概率与风险模型核心
- `app.js`：临时安全入口，避免页面报错

说明：完整自动同步版本因为平台安全检查限制，部分前端自动分析代码无法直接写入仓库。本次已另外生成完整代码压缩包，可下载后手动上传或后续分模块继续改。

## V2 目标结构

- 自动同步赛事
- 自动读取球队近期状态
- 自动整合伤停、停赛、休息时间、天气、海拔、场地、公开新闻信号
- 输出概率、风险等级、模型信心、因子解释

## 数据源建议

推荐部署到 Netlify，因为 GitHub Pages 不能安全隐藏 API Key。

可配置的数据源：

- football-data.org：赛程、球队、排名、比赛数据
- The Odds API：公开市场价格与多地区数据
- OpenWeather：天气、风速、湿度、降雨
- API-Football / Sportmonks / Opta：伤停、停赛、阵容、更多商业数据

## 重要口径

本项目只做公开数据建模、风险评估和内容讨论，不承诺命中，不构成投注建议。

更衣室关系、教练私人关系、球员私人关系等内容，不能伪造成确定事实，只能通过公开新闻、发布会、可信记者报道做低权重舆情信号。
