# 鲸喜足球预测助手

一个手机端可用的足球赛事预测 H5 小程序原型。

## 功能

- 赛事录入
- 八维评分模型：实力、攻防、身价、历史、环境、天气、裁判、战意
- 自动输出胜平负概率
- 自动判断总进球主推区间
- 自动生成购买方案
- 赛后复盘
- 本地历史记录

## 使用方法

直接打开 `index.html` 即可运行。  
也可以上传到 GitHub Pages / Netlify / Vercel。

## 部署到 GitHub Pages

1. 新建 GitHub 仓库：`football-prediction-assistant`
2. 上传本项目所有文件
3. 进入仓库 Settings → Pages
4. Source 选择 `Deploy from a branch`
5. Branch 选择 `main` / root
6. 保存后等待部署链接生成
