## 下一阶段

### Phase 1：内部可用版

目标：不买域名，不接复杂商业接口，先让系统真正可用。

任务：

- 建立服务器后端接口
- 把演示数据迁移为服务器数据
- 增加后台数据文件
- 增加一键更新脚本
- 增加赛事数据导入功能

### Phase 2：基础自动数据

目标：让程序自动拿基础数据。

任务：

- 接天气数据
- 接基础赛程数据
- 接球队近期比赛数据
- 接排名数据

### Phase 3：专业数据

目标：接入更完整的专业足球数据。

任务：

- 接球员数据
- 接阵容数据
- 接缺阵数据
- 接停赛数据
- 接教练数据
- 接战术和阵型数据

### Phase 4：信号系统

目标：把公开信息转成模型信号。

任务：

- 新闻来源采集
- 公开采访采集
- 可信度评分
- 信号权重计算
- 模型解释输出

## 用户必须本人完成的事情

- 注册第三方数据平台账号
- 购买付费数据套餐
- 输入支付信息
- 提供 API Key
- 腾讯云账号登录或安全验证

我不能代替用户完成账号支付、实名认证、安全验证或查看隐藏密钥。
