# 鲸喜直播猜词互动系统

这是第一版可运行原型，定位为直播间弹幕互动小游戏。

## 已完成功能

- 语义猜词主界面
- 倒计时与进度条
- 随机题库
- 手动弹幕输入测试
- 答案匹配度计算
- 猜中自动加分
- 排行榜本地保存
- 礼物互动模拟
  - 点赞：加分
  - 小心心：加时
  - 大啤酒：随机换词
  - 鲜花：双倍得分
- 本地词库添加
- 移动端/电脑端自适应

## 运行方式

直接打开：

`live-guess-word/index.html`

如果 GitHub Pages 已开启，可访问：

`https://ljivfnaj1b50.github.io/football-prediction-assistant/live-guess-word/`

## 当前版本说明

当前是 MVP 演示版，直播评论通过手动输入模拟。后续需要接入抖音直播弹幕时，应使用合规的弹幕监听服务或主播授权工具，不做刷量、挂机、绕过平台限制等违规功能。
